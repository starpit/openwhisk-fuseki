# openwhisk-fuseki

A simple [OpenWhisk](https://github.com/openwhisk/openwhisk) wrapper
around [Apache
Fuseki](https://jena.apache.org/documentation/fuseki2/).

## Prerequisites

 1. [install the `wsk` CLI](https://bluemix.net/openwhisk/cli)
